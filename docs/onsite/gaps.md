## 1. 規格体系

### 試験規格
- 🟢 JIS / ASTM / ISO の基本構造、引張・硬度・衝撃・疲労・クリープの代表規格番号
- 🟡 ASME（圧力容器系、Section II など）の体系
- 🟡 EN（欧州規格）と ISO の対応関係
- 🔴 **AMS (SAE Aerospace Material Specifications)** — 航空機エンジン材料の主要規格群、Ti / Ni 基系の AMS 4xxx〜5xxx 番台
- 🔴 **FAR Part 25** — 航空機型式証明、材料ばらつきの A/B/S basis 値の出処
- 🔴 **JIS H 7700 系** — 航空宇宙用チタン合金の試験要求

### 業界横断
- 🟢 ISO 9001 (QMS) の構造
- 🟡 **ISO/IEC 17025** (試験所認定) の要求事項、特に測定不確かさの記述義務
- 🔴 **NADCAP** (National Aerospace Defence Contractors Accreditation Program) — 航空宇宙系試験所の必須認定
- 🔴 FDA 21 CFR Part 11 (電子記録 / 電子署名) — 医療機器材料案件で要求される可能性

### 切削系
- 🟢 ISO 3685 (工具寿命試験 / VB 閾値)
- 🟡 ISO 8688 (フライス工具寿命)
- 🔴 ISO 13041 (NC 工作機械の精度試験) — 機械の動的剛性評価
- 🔴 **VDI 3441 / VDI 3423** (工作機械精度・信頼性) — ドイツ規格、欧州顧客で参照される

### 学習計画
- AMS 規格の番号体系を 1 時間で総覧（SAE のサイトで概要）
- ISO 17025 の章立てを 30 分で読む（測定不確かさだけでも掴む）
- 現場で「NADCAP 取ってますか？」を最初の質問に入れる

## 2. セキュリティ・輸出管理

### 想定リスク
- 🔴 **ITAR (International Traffic in Arms Regulations)** 対象製品（軍用エンジン部品など）の場合、非米国市民へのデータ開示禁止
- 🔴 **EAR (Export Administration Regulations)** — 民生品でも輸出管理該当の可能性
- 🔴 **外為法（外国為替及び外国貿易法）** — 日本側の輸出管理、リスト規制 / キャッチオール
- 🟡 **クラウドサービスの所在地問題** — データが海外サーバを通過する場合の規制
- 🟡 **暗号化要件** — TLS 1.2+ / AES 256 等の最低基準

### 設計影響
- Vercel / Supabase 等のクラウドが NG の場合、Docker + オンプレ Postgres 版を用意できる必要
- API キーの取り扱い（環境変数 / KMS / Vault）
- ログから機密情報を除外する仕組み
- 監査証跡の保持期間 / 改ざん防止

### 学習計画
- 自社（雇用先）の輸出管理ポリシーを読む
- クライアント側の規制状況を初回ヒアリングで確認
- Docker 版 / オンプレ版の概念設計を先に書いておく

## 3. 既存システム連携

### 想定する既存システム
- 🟡 **LIMS (LabWare / Matrix Gemini / Sample Manager / 社内製)** — どれかは使われている前提
- 🟡 **SAP / 自社 ERP** — 案件・売上・原価の連携先
- 🟡 **CAD/CAM (NX / CATIA / hyperMILL / TopSolid)** — 試験片図面 / 加工指示
- 🔴 **MES (Manufacturing Execution System)** — 製造現場の実績管理
- 🔴 **試験機メーカ別ソフト** (Instron BlueHill / Zwick testXpert / MTS TestSuite 等) — 個別の出力フォーマット

### 連携要件想定
- 試験機ソフト → CSV / TDM / TDMS / RTD 等の出力フォーマット
- LIMS → REST / SOAP / ファイル取込（XML / CSV）
- ERP → 案件番号の同期、月次集計
- CAD/CAM → STEP / IGES での図面取込（多分やらない）

### 学習計画
- 主要 LIMS の API 仕様（公開範囲）をざっと見る
- 試験機メーカの代表的な出力フォーマットを把握
- 「Matlens は連携層に徹する / 置換しない」スタンスを最初の宣言に含める

---

## 4. 試験報告書の法定要件

### 想定要件
- 🟡 ISO/IEC 17025 7.8 — 報告書必須記載事項（測定方法、結果、不確かさ、判定基準）
- 🟡 顧客指定テンプレート（多くの航空 / 自動車系メーカは独自フォーマット）
- 🔴 電子署名・タイムスタンプの法的有効性（電子署名法 / eIDAS）
- 🔴 改ざん防止（PDF/A、長期保存形式）

### Matlens の現状
- Markdown 本文は書ける
- PDF/DOCX 出力は未実装（Phase 4）
- 電子署名・版管理は未対応

### 学習計画
- ISO 17025 7.8 の項目を 30 分で読む
- 顧客サンプルレポートを初回ヒアリングで提供してもらう
- 電子署名は「Stage 2 で要件確定後に決定」と明確に告白する

## 5. 切削加工の現場知

### 把握済み
- 🟢 Vc / f / ap / ae の意味と単位
- 🟢 工具材種（HSS / 超硬 / コーティング / セラミクス / CBN / PCD）の特徴
- 🟢 Taylor / Kc / Stability Lobe の数式（PR #66 で実装）
- 🟢 VB / Ra の標準閾値（ISO 3685 / ISO 4287）

### 曖昧
- 🟡 **5 軸同時制御** の運動学（A/B/C 軸の組合せ、姿勢補正）
- 🟡 **切削温度測定** の実務（埋込熱電対、赤外線、切削抵抗からの推定）
- 🟡 **残留応力測定** の手法（X 線回折、穿孔法、incremental hole-drilling）
- 🟡 **白層・加工変質層** の評価（断面組織、硬度プロファイル）

### ほぼ未知
- 🔴 **超音波振動切削（UAM / UVT）** — 難削材で使われる
- 🔴 **クライオ切削（液体窒素 / CO2）** — Ti 合金の温度抑制
- 🔴 **MQL（最少量潤滑）の実務** — 環境配慮で増加中
- 🔴 **ハイブリッド製造（DED + 切削）** — 積層 + 切削の組合せ

### 学習計画
- 5 軸の基本（B-table type / B-head type）を 1 時間で
- 残留応力測定の代表手法 3 つを概念把握
- 「白層」「加工変質層」を画像で見れる事例集を探す
- ヒアリングで「特殊加工法やってますか？」と振る

## 6. 業界別ドメイン知識

### 航空機エンジン（最有力ターゲット）
- 🟡 主要材料: Ti-6Al-4V, IN-718, IN-625, René 41, Waspaloy, CMSX-4
- 🟡 PAM (Plasma Arc Melting) / VAR (Vacuum Arc Remelting) の溶解工程
- 🔴 PQ (Premium Quality) の判定基準と試験項目
- 🔴 NDE (Non-Destructive Examination) の航空業界特有の要件
- 🔴 主要 OEM (GE / Pratt & Whitney / Rolls-Royce / IHI / 三菱重工) の調達基準

### 自動車（次点）
- 🟡 IATF 16949、PPAP、APQP の概要
- 🔴 Tier 1 サプライヤーへの納入要件

### 学習計画
- IHI / 三菱重工の公開技術論文を 2〜3 本読む
- PQ / NDE の入門記事を漁る
- 「航空 / 自動車 / それ以外」のどれが主かを最初に確認

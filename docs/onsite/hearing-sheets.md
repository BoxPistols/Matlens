# 画面別ヒアリングシート

各 Signature Screen ごとに **「見せるもの → 確認したい仮説 → 想定回答パターン → メモ」** の 4 列で整理。
現場ヒアリングではこの順で口頭展開すると沈黙が生まれにくい。

凡例: ✅ 仮説が当たった | ⚠️ 部分的にズレ | ❌ 外れ — 結果はメモ欄に書き込む。

# Matlens 5 分オンボーディング

> Matlens を初めて触る人向けの 5 分動線。**MaiML を中核として、Import → 確認 → Export → Inspect** の 1 周をデモします。

## 前提

- ブラウザで Matlens を開いた直後（`/` または `#/dash`）
- `src/mocks/fixtures/` の決定論的シードが読み込まれている
- 所要時間: 約 5 分

---

## ステップ 1: MaiML Studio へ移動（30 秒）

サイドバー左の **「コア」セクション → MaiML Studio**（CORE バッジ付き）をクリック。

MaiML Studio Hub が開き、5 つのカード（インポート / エクスポート / インスペクト / バリデート / Diff）と「MaiML とは」コラムが表示されます。

> Matlens のすべての MaiML 操作はここから始まる、という設計を体感する箇所です。

---

## ステップ 2: サンプル MaiML をインポート（1 分）

1. **「インポート」カード** をクリック
2. drop zone に `.maiml` ファイル（後述: 「サンプルファイルの作り方」を参照）をドロップ
3. プレビュー画面で以下を確認:
   - 取込件数（先頭 10 件のテーブル）
   - 警告 0 件（出力日時 / 出力元）
4. 右下「N 件を取り込む」ボタンで commit

> drop → preview → commit の **3 段階導線** によって、即反映による事故を防いでいます。

---

## ステップ 3: 取り込んだ材料を確認（1 分）

1. 取込完了画面の **「材料データ一覧へ」** ボタンをクリック → `#/list`
2. 取り込んだ材料が一覧に追加されている
3. 任意の材料行をクリック → 詳細ページ
4. 詳細ページ右上の **「MaiML エクスポート」** ボタンで往復確認（プレビュー → ダウンロード）

> これで 1 材料の round-trip（インポート → 表示 → 再エクスポート）が完了。

---

## ステップ 4: 試験マトリクスから試験集合を MaiML 出力（1.5 分）

1. サイドバー **「データ → 試験マトリクス」** へ移動
2. 任意のセル（材料 × 試験種別）をクリック → 右サイドバーに該当試験一覧
3. 一覧上部の **「全件 MaiML」** ボタンで一括エクスポート、または各試験行の **「MaiML」** ボタンで単一エクスポート
4. プレビューで内容確認 → ダウンロード

> セルから外部 LIMS / 顧客 OEM 引渡し用の MaiML を切り出せる導線。

---

## ステップ 5: ダウンロードした MaiML をインスペクト（1 分）

1. サイドバー **「コア → MaiML Studio → インスペクト」**
2. ダウンロードした `.maiml` ファイルを drop zone にドラッグ
3. XML が整形表示される
4. 検索ボックスに `provenance` や `result` を入力 → ハイライト表示
5. 統計（バイト / 行数 / 要素数）でドキュメント全体感を把握

> XML の中身を「目で確認できる」状態にすることが、現場でのトラブル切り分けの第一歩。

---

## サンプルファイルの作り方

開発環境では決定論的シードからの出力で十分:

1. `#/list` の任意材料 → 詳細 → 「MaiML エクスポート」 → ダウンロード
2. または `#/pjlist` の任意案件 → 「MaiML エクスポート」 → ダウンロード

これらをサンプルとしてインポート / インスペクトに使えます。

---

## 次の動線

5 分動線の後、興味に応じて:

- **検索 → AI チャット**: 取込済材料に対する RAG 質問
- **解析 → ベイズ最適化**: 材料特性の探索
- **工程 → ペトリネット**: 試験ワークフローの可視化
- **可視化 → 結晶構造 3D**: 物質の結晶格子表示

# ADR 0007 — シミュレーション Tier 設計と「参考値」開示方針

| 項目 | 内容 |
|---|---|
| 状態 | 採用 |
| 決定日 | 2026-04-12 |
| 関連 issue | #15 (Phase E), #27 (B-3) |

## 背景

金属材料の物性予測には精度の異なる複数のアプローチがある。
「どこまで信じてよいか」をユーザーに明示しないと、推定値を実測値と混同するリスクがある。

## 決定

シミュレーション機能を 3 Tier に分類し、各 Tier ごとに開示レベルを明確化する。

### Tier 1: 経験式予測 (実装済み — PR #29)

| 式 | 入出力 | 精度 | 開示 |
|---|---|---|---|
| Hall-Petch | 粒径 → 降伏応力 | 定性的 | 「参考値」バッジ + 式の表示 |
| Larson-Miller | 温度×時間 → クリープ寿命 | 桁推定 | 同上 |
| JMAK | 時間 → 変態分率 | 傾向 | 同上 |
| ROM | 体積分率 → 弾性率 | 上下界 | Voigt/Reuss 両方を表示 |

- **入力**: ユーザーがパラメータを直接入力
- **開示**: 数式を折りたたみパネルで開示、「参考値」バッジ (amber) を常時表示
- **根拠**: 式の出典 (著者 + 年) を明記

### Tier 2: 可視化中心のシミュレーション (未実装)

- 熱処理 CCT/TTT 図
- 酸化膜成長
- 疲労き裂進展

現状は Phase 3 後半または Phase 4 のスコープ。

### Tier 3: ベイズ最適化 (実装済み — PR #22, #30)

| 手法 | 精度 | 開示 |
|---|---|---|
| GP 回帰 (1D/2D) | データ依存 | ±2σ 不確実性バンド + EI 値 |
| Expected Improvement | 提案のみ | 「次に試すべき」の根拠を数値で表示 |

- **入力**: 既存 DB からデータ自動抽出
- **開示**: GP 平均曲線 + ±2σ バンドをグラフ表示、EI 値を KPI カードで明示
- **根拠**: アルゴリズム解説パネル (GP / EI の数式と直感的説明)

## 「参考値」開示の原則

1. **全ての推定値に provenance を付与する**: `provenance: 'ai' | 'simulation'` フィールド (Material 型に追加済み)
2. **推定値は点線下線で示す** (UI — 未実装だが DetailPage の provenance バッジで代替)
3. **式・モデル・前提条件を折りたたみで常に参照可能にする**
4. **推定値と実測値を混在表示する場合は色分けする** (ai=紫, simulation=橙)
5. **エクスポート時に provenance を MaiML XML に含める** (実装済み — maiml.ts)

## 代替案

- 推定値を別テーブルに分離する → 却下。実測値との比較こそが価値。混在表示 + バッジで区別する方が実用的。
- 推定値を表示しない → 却下。「次に何を実験すべきか」の提案にはモデル出力の表示が必須。

## 影響

- SimulationPage: 「参考値」バッジ + 式の表示 (実装済み)
- BayesianOptPage: EXPERIMENTAL バッジ + ±2σ バンド (実装済み)
- DetailPage: provenance バッジ (実装済み)
- MaterialFormPage: provenance フィールド入力 (ステップウィザード Step 2 に統合済み)

# ドメインリサーチメモ

現場入り前の自学用。公開情報ベースの概観を自分の言葉で整理し、
ヒアリング時に「何を聞くべきか」を明確にするための素材。

実数値・顧客固有情報は含めない。業界の骨格・規格の体系・
実務で登場する用語の地図を作るための**学習ドラフト**。

## 書き方ルール

- 公開情報のみ。実案件の数値・顧客名は書かない
- 「現場で聞くこと」を各メモの末尾に必ず添える（ヒアリング連動）
- Confidence（中 / 高 / 低）を明記
- 実データ・顧客対話で上書きされる前提で Draft 扱い

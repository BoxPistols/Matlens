## なぜ押さえるか

- 求人 MUST に「切削抵抗、強制振動、びびり振動、**切削温度、残留応力**」が列挙
- これらは **測定が難しい / 間接推定が多い**ため、手法の選択肢と限界を押さえておく必要
- 加工変質層（白層・硬化層）は航空部品の疲労寿命に直結 → 評価手法の知識が効く

## 1. 切削温度の計測

### 1.1 測定箇所のジレンマ

- 切削温度が最も高いのは **せん断帯 / 工具刃先近傍**（1000 ℃超）
- しかし刃先は **高速で移動 + 冷却材 + 切屑に覆われる** → 直接測定が困難
- 工具側・ワーク側・切屑側・雰囲気側のどこで測るかで得られる物理量が違う

### 1.2 手法一覧

| 手法 | 原理 | 精度 | 実装難度 | 備考 |
|---|---|---|---|---|
| **埋込熱電対（ワーク側）** | K 型 / T 型熱電対をワーク内に埋設 | ±2 ℃ | 高（穴加工必要） | 応答 0.1〜1 s |
| **埋込熱電対（工具側）** | インサートに穴を開けて熱電対 | ±5 ℃ | 高 | 工具寿命に影響 |
| **tool-work 熱電対** | 工具とワークを熱電対対として起電力測定 | ±10 ℃ 程度 | 中 | 絶縁・校正が課題 |
| **赤外線カメラ（IR）** | 刃先 / 切屑を撮像 | ±5〜20 ℃ | 中 | 切屑 / 冷却材で視野遮られる |
| **赤外線単点（パイロメータ）** | 狭視野・高速応答 | ±10 ℃ | 中 | サブミリ秒応答可 |
| **切削抵抗からの推定** | Kienzle 係数経由の発熱推定 | ±20〜50 ℃ | 低 | 絶対値よりトレンド向き |
| **結晶組織変化からの事後推定** | 切屑 / 加工面の組織を SEM で観察 | 〜ゾーン識別 | 高 | リアルタイム不可 |

### 1.3 実務での典型構成

- **ドライ切削 / MQL**: IR カメラ（刃先を斜めから狙う）+ 赤外線スポット
- **ウェット切削**: 冷却材で IR が効かない → 埋込熱電対 or tool-work
- **研究室ベンチ**: 複数手法を同時使用（IR + 熱電対 + 切削抵抗 → 相関解析）

### 1.4 Matlens への示唆

- `CuttingForceSample` 型と同じ粒度で `TemperatureSample` を追加する余地
- Kienzle モデル（`kcForceModel.ts`）から **発熱推定値**を計算するユーティリティ拡張
  - $P = F_c \cdot v_c$ → 発熱に概ね等しい（機械効率を除いて）
- 表示は波形ビューアで抵抗・温度を **同期マルチチャネル**に

## 2. 残留応力測定

### 2.1 何を知りたいか

- 加工後の表層に残る残留応力は **疲労寿命に直接影響**
- 圧縮残留応力（-MPa）: 疲労寿命を延ばす（ショットピーニング効果）
- 引張残留応力（+MPa）: 疲労寿命を縮める（クラック伝播促進）
- **切削条件で符号・大きさが変わる** のが研究対象

### 2.2 手法一覧

| 手法 | 原理 | 分解能 | 破壊性 |
|---|---|---|---|
| **X 線回折法 (XRD / sin²ψ 法)** | 格子面間隔の歪みから応力推定 | 表層 5〜30 µm | 非破壊 |
| **穿孔法 (hole-drilling / ASTM E837)** | 小穴を空け、周辺のひずみ緩和からチェック | 表面〜1 mm 程度 | 半破壊 |
| **incremental hole-drilling** | 穿孔を段階的に行い、深さ方向プロファイル | 1.5 mm 以内 | 半破壊 |
| **輪郭法 (contour method)** | 切断後の変形量から断面応力を再構成 | 断面全域（mm 精度） | 破壊 |
| **中性子回折** | 貫通型の回折で内部応力 | 数 mm 深さ | 非破壊 |
| **放射光 X 線** | シンクロトロン光源で高精度 XRD | 数 mm 深さ | 非破壊 |
| **磁歪法（バルクハウゼンノイズ）** | 磁性材限定、強磁性体の磁化特性 | 表層数十 µm | 非破壊 |

### 2.3 実務での位置付け

- **XRD (sin²ψ 法)**: 航空現場で最も普及。持込型の測定機もある
- **incremental hole-drilling**: 深さ方向プロファイルが必要な研究向け
- **輪郭法**: 大型鍛造品の断面応力分布を見る研究向け
- **中性子 / 放射光**: 特殊施設のみ（J-PARC / SPring-8）

### 2.4 切削条件との相関（文献より）

- 高速切削（高 Vc）: 表層温度上昇 → 引張残留応力になりやすい
- 切込み深い: 機械的歪み支配 → 圧縮になりやすい
- 工具摩耗進行: 引張側にシフト
- MQL / クライオ: 温度抑制 → 圧縮保持

### 2.5 Matlens への示唆

- `Test.category = 'residualStress'` に対応する `resultMetrics` 項目を拡充
  - 例: `residualStressMPa_depth_0um`, `residualStressMPa_depth_50um` ... のようなプロファイル
- 切削条件エクスプローラで **条件 vs 残留応力** の散布図を追加できる

## 3. 加工変質層・白層

### 3.1 何を観察するか

- **加工変質層 (Affected Layer)**: 切削熱・塑性変形で元の組織が変化した表層
- **白層 (White Layer)**: 急速加熱急冷による超硬化層（Ni/Ti で特徴的）
- **塑性流動層**: ミクロな塑性変形が積層された層
- これらは断面研磨 → エッチング → 光学顕微鏡 / SEM で観察

### 3.2 測定手法

| 手法 | 観察対象 | 典型深さ |
|---|---|---|
| **光学顕微鏡 + エッチング** | 白層・流動層の目視確認 | 数十 µm |
| **SEM（走査電子顕微鏡）** | 組織微細構造 | nm 級 |
| **EBSD（電子後方散乱回折）** | 結晶方位分布 | nm 級 |
| **マイクロビッカース硬度プロファイル** | 深さ方向硬度 | 50 µm 刻み |
| **ナノインデンテーション** | µm オーダー硬度 | sub-µm |

### 3.3 白層の問題

- 非常に硬い（元母材の 2 倍以上もある）
- クラック発生起点になり、**疲労寿命を大幅に低下**
- Ni 基超合金（Inconel 718）+ 超硬工具の切削で発生しやすい
- 対策: 切削条件最適化 + CBN 工具使用 + MQL / クライオ

### 3.4 Matlens への示唆

- `DamageFinding.type = 'whiteLayer' / 'heatAffectedZone' / 'plasticFlow'` を追加
- 損傷ギャラリーに **断面組織画像** を登録する用途
- 切削条件と白層発生率の相関分析
